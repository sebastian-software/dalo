# v0.3.0 Release Notes

Date: 2026-06-29
Changelog: [`0.3.0`](../../CHANGELOG.md)

`v0.3.0` delivered the V1.1 catalog and instruction-pack layer on top of the V1 local/team loop.

## Highlights

- Catalog sources with `source add-catalog`, `source inspect`, and `source select`.
- Source locks that pin catalog commits, selected skill refs, and inventory snapshots.
- Read-only catalog drift checks with `source refresh --check`.
- Same-catalog required-closure expansion with approval and linkability preflight.
- Instruction packs rendered into explicit managed blocks with `instructions enable` and removed with `instructions disable`.
- Instruction pack discovery and `instructions list`.
- Topic-overlap warnings in `status` and `doctor`.

## Safety and Polish

- Catalog and instruction edge cases were hardened after the initial V1.1 feature work.
- Later v0.4 planning work continues the same safety theme: failure-path correctness, lock handling, docs, distribution, and website accuracy.
