# v0.1.0-rc.1 Release Notes

Status: Historical draft release candidate
Date: 2026-06-24

`v0.1.0-rc.1` is the first Dalo release candidate for the V1 local/team skill loop.

## Included

- Store initialization and local private source.
- Target detection and linking for supported directory-based agents.
- Team Git source add/list/priority.
- Inventory scanning for `SKILL.md` skills.
- Deterministic resolution with approvals, shadowing, and local overrides.
- Safe directory symlink materialization.
- Dirty team source blocking.
- Resolved user lock writing and lock drift reporting.
- Unmanaged skill discovery.
- Copy-first adoption and optional owned-symlink replacement.
- Minimal resolve helpers.
- Doctor diagnostics.
- Linux and macOS CI.

## Deferred at the time

Some items in this historical RC list shipped later. See `v0.2.0.md`, `v0.3.0.md`, and `../../CHANGELOG.md` for current release history.

- Catalog and external source selection. Catalog sources shipped in v0.3.0; external source declarations remain planned.
- Instruction packs. Managed instruction packs shipped in v0.3.0.
- Autosync installer.
- Promote and PR creation.
- Source refresh PRs.
- Homebrew/Cargo distribution. Cargo and GitHub release artifact automation was wired after the MIT license file landed.
- Windows support.

## Validation

The release candidate gate is:

```sh
cargo fmt --check
cargo test
cargo clippy --all-targets --all-features -- -D warnings
cargo build --release
git diff --check
```

The test suite uses local temporary directories and local Git repositories only.
