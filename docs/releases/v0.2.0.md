# v0.2.0 Release Notes

Date: 2026-06-25
Changelog: [`0.2.0`](../../CHANGELOG.md)

`v0.2.0` delivered the V1 local/team skill loop: store setup, target linking, source management, resolution, safe materialization, adoption, and diagnostics.

## Highlights

- Rust CLI and reusable library scaffold.
- `dalo init` store layout with TOML config, state, approvals, and user lock.
- Target registry and `target detect`/`link`/`unlink`.
- Local and team Git sources with clean refresh and dirty-source blocking.
- Inventory scanner for `SKILL.md` skill directories.
- Deterministic resolver with priority, approvals, shadowing, and local overrides.
- `status` and `sync` for resolving and materializing directory symlinks.
- Copy-first adoption flow plus minimal `resolve` repair helpers.
- `doctor` diagnostics for store, Git, targets, symlinks, dirty sources, pending approvals, and unmanaged blockers.

## Follow-Up Work

- Catalog source selection, catalog drift, and same-catalog required closures shipped later in v0.3.0.
- Instruction pack managed blocks and topic-overlap warnings shipped later in v0.3.0.
- Distribution automation was wired after the root MIT license file landed.
